== Description ==

Connect your Google Webmaster Tools to your admin area.

Queries, Top Pages, Keywords, Links to your Site, and Internal Link Stats.

Great plugin for Seo!

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload 'mdp-google-webmaster-tools' to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= Does this connect to my google webmaster tools account? =

Yes, all you need to do is enter your email address and password.

= How often will this update? =

Once per hour.
