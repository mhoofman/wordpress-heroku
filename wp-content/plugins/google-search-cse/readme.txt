Add a Custom Google Site Search with no setup. Just drag and drop a widget and you're done.

== Description ==

Easily setup a full text site search powered by Google.

Normally to setup a Google Custom Search on your site you have to sign up on Google and create the search index. This plugin will do it automatically using Google's [Linked Custom Search Engines](http://www.google.com/cse/docs/cref.html)

*   [View a Live Demo Â»](http://www.seedprod.com/blog/ "SeedProd Demo")


### Features:

*   Works with any WordPress Theme
*	No need to setup a Google Custom Search
*	No Google account required 
*	Search results show up on the default search result page.
*	Just add the "Google Site Search" widget to your theme.
*	Does not replace the default WordPress search. It runs along with it. Use both if you want.

This plugin is built and maintained by [SeedProd](http://www.seedprod.com/)

== Installation ==

Installation

1. Click **Plugins** > **Add New** > search for **WordPress Google Custom Search**
1. Activate the plugin through the **Plugins** menu in WordPress
1. Click **Appearance** > **Widgets** > Drag the **Google Search** widget to a widget area.


== Frequently Asked Questions ==

*  Why are ads showing on my search results?

This is because Google puts ads on search results by default. [Read more](http://www.google.com/cse/docs/cref.html)

* No search result are displaying

This mean that no pages from you site are in Google index. Learn how to [submit your site](http://www.google.com/submityourcontent/).
